==PETUNJUK==

Sebelum menjalankan TokoEmas.jar :
	1. Pastikan OS sudah terinstall Java
	2. Extract TokoEmas.zip (pastikan semua konten ada di folder yang sama)
	2. Jalankan XAMPP / WAMP terlebih dahulu.
	3. Pastikan folder "lib" dan TokoEmas.jar berada pada 1 folder
	4. Install Database menggunakan TokoEmas.sql
		4.1 Nyalakan browser dan ketikan localhost/phpmyadmin
		4.2 Pilih tab "Import"
		4.3 Pada bagian File to Import, silahkan spesifikasikan letal TokoEmas.sql
		4.4 Klik "Go"
	5. Jalankan

Username dan password bersifat CASE SENSITIVE / huruf kapital dan huruf kecil berpengaruh
